Internal API Reference Guide

This page provides an overview of known functional endpoints and the corresponding actions available through the Nebula Dynamics system. Use responsibly and with authentication where required.

Endpoint Method Description Authentication
/telemetry/submit POST Submits spacecraft telemetry data. Expected parameter: node. Optional (flagged if malformed XML)
/admin GET, POST, OPTIONS Administrative dashboard access. None
/internal/notes GET Contains debug notes. Use set_debug URI to enable functionality. None
/progress GET Visualizes rocket mission progress. Primarily static output. None
/help GET This page. Lists available API methods and usage for developers and operatives. None
/report/render POST Used to render PDF reports. Only accepts HEX addresses (e.g. 0xHEX VALUE) when using URLs. Accepts XML requests with data. None
/api/submitlog/ POST Use the data value "entry=VALUE" to write logs to the system. None
